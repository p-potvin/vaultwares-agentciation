# vaultwares-agentciation

Reusable base for multi-agent coordination and communication using Redis.

## Contents
- extrovert_agent.md: Parent agent design and rules
- redis_coordinator.py: Redis-based coordination logic (heartbeat, status, reminders)
- agent_base.py: Base class for agents (for inheritance)
- enums.py: Status enums and shared constants

## Usage
Import as a submodule in future projects for robust agent orchestration.
