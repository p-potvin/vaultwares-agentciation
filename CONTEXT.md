# VaultWares Agentciation — Context

This repository is the **source of truth** for reusable, cross-project agent infrastructure:

- Custom agents and agent definitions
- Reusable skills (`SKILL.md` packages)
- Shared coordination protocols (dispatcher/task lifecycle)
- Tooling to sync agent instructions across IDEs/assistants

Project-specific instructions (per-repo `AGENTS.md`, `CONTEXT.md`, project runbooks) must remain in the project repository.

## Docs Location

Most reference documents live under `docs/`. Root-level `*.md` files that previously held large documents may exist as thin pointers for compatibility.
